1) O que é Programação Orientada a Objetos (POO)?

Programação Orientada a Objetos é um paradigma de programação baseado no uso de objetos, que representam entidades do mundo real.
Na POO, o sistema é organizado em classes e objetos, que reúnem dados (atributos) e comportamentos (métodos).

2) Quais as vantagens da POO em relação à programação estruturada?

Melhor organização do código
Maior reutilização de código
Facilidade de manutenção
Melhor controle e proteção dos dados
Mais adequada para sistemas grandes

3) Defina o conceito de objeto no paradigma POO

Objeto é uma instância de uma classe.
Ele possui:
Estado (atributos)
Comportamento (métodos)
Identidade (é único no sistema)

4) Explique os 4 principais fundamentos da POO
   
Encapsulamento:
Protege os dados do objeto, permitindo acesso apenas por métodos definidos.
Abstração:
Representa apenas as características essenciais de um objeto, escondendo detalhes desnecessários.
Herança:
Permite que uma classe herde características de outra, reutilizando código.
Polimorfismo:
Permite que um mesmo método tenha comportamentos diferentes dependendo do objeto.
